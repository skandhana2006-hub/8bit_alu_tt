# 8-bit ALU TinyTapeout Project

## Description
This project implements an 8-bit Arithmetic Logic Unit (ALU) using Verilog for TinyTapeout.

## Operations Supported
- Addition
- Subtraction
- AND
- OR
- XOR
- NOT
- Left Shift
- Right Shift

## Inputs
- ui_in  -> Input A
- uio_in -> Input B

## Output
- uo_out -> Result Output
